# Published result bundles

This directory is reserved for sanitized, content-addressed analysis bundles admitted under
`docs/leaderboard-governance.md`. The final release workflow expects the primary registered study
at `results/published/primary` and refuses to publish without a verified, manifest-complete,
non-mock result bundle.

Raw Inspect logs, provider payloads, credentials, private annotation keys, and unreviewed
development outputs do not belong here.
